# Portfolio BTS SIO SISR

Site statique HTML/CSS représentant un portfolio pour un étudiant en BTS SIO option SISR.

## Structure
- `index.html` : page principale avec le menu et les sections.
- `css/style.css` : styles avec palette bleue et gris clair.
- `projets-entreprise.html` : page séparée détaillant les projets réalisés en entreprise.

## Navigation
- Accueil
- À propos
- TCS
- BTS SIO
- Projets
  - "Mes projets en entreprise" ouvre la page `projets-entreprise.html`
  - Projets scolaires E6
- Veilles
  - Veilles technologiques
  - Veilles juridiques

## Lancer
Ouvrir `index.html` dans un navigateur.
